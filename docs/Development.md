# 💻 개발 가이드 (Development)

이 문서는 **Bokaro Hive**의 기술적인 구조와 빌드 환경에 대해 설명합니다.

## ⚙️ 개발 환경
*   **프로젝트명**: Bokaro Hive
*   **사용 언어**: C
*   **컴파일러**: Raylib Compiler (MinGW 기반)
*   **사용 라이브러리**: Raylib (그래픽, 입력, 오디오 처리)
*   **운영체제**: Windows 10/11

## 🔨 빌드 방법
본 프로젝트는 Raylib 컴파일러 환경을 기준으로 빌드됩니다.

1.  **VS Code 빌드**:
    *   `.vscode/tasks.json`에 설정된 빌드 태스크를 활용합니다.
    *   `Ctrl + Shift + B`를 눌러 현재 소스 코드를 컴파일합니다.
2.  **CLI (커맨드 라인) 빌드**:
    *   Raylib 컴파일러 전용 터미널을 실행합니다.
    *   프로젝트 루트 디렉토리에서 `make` 명령어를 입력합니다.
    *   컴파일 완료 후 생성된 실행 파일(`.exe`)은 `bin/` 폴더에 배치됩니다.

## 📂 프로젝트 구조
*   **소스 코드**: C 언어로 작성되었으며, 게임의 메인 루프와 로직을 담당합니다.
*   **그래픽 및 사운드**: Raylib 라이브러리를 통해 리소스를 로드하고 화면에 출력합니다.
*   **에셋 경로**: 실행 시 `assets/` 폴더 내의 이미지와 폰트 데이터를 참조합니다.

## ⚠️ 주의 사항
*   컴파일 및 실행 시 `raylib.dll` 파일이 `bin/` 폴더 내에 반드시 존재해야 합니다.
*   새로운 리소스를 추가할 경우 `assets/` 폴더 구조를 유지해야 프로그램이 정상적으로 데이터를 불러옵니다.
