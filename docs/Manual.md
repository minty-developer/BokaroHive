## 🛠️ 문제 해결 (FAQ)
**Q. 소리가 들리지 않아요.**
A. 게임 설치 폴더 내에 `bin` 폴더의 오디오 드라이버 DLL 파일들이 있는지 확인해 주세요. (~v1.0.0까지는 소리가 없습니다.)

**Q. 글자가 깨져서 보입니다.**
A. 시스템에 폰트가 설치되지 않았을 수 있습니다. 설치 프로그램(`Setup.exe`)을 다시 실행하여 '폰트 자동 등록'을 완료해 주세요.
